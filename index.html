In the navbar above the last four links are API links.

  • openWeatherMaps API

    • Has a single day forecast and a seven day forecast.

  • NASA API!

    • That uses the APOD or Astronomy Picture of the Day API. It displays three photos that will open up into larger photos in another tab if you click on them.

  • Farmers Market API!

    • This queries a government farmers market database that kinda sucks. It returns a string that has the miles away at the beginning and name of the event at the end. Like this: 2.4 Downtown Los Angeles Farmers Market Using this API required parsing the strings to get those values separated, and I also generated an anchor tag that auto-fills a google search with the name of the event because there is no URL or any other information included.

  • eventBrite API

    • Unfortunately for me, I did a bunch of work to parse a sample object and then found out at the end that this was a deprecated (no longer existing) API. So I use a demo object to demonstrate it.
